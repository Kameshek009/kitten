# 🎉 Открытка для поздравлений

Полнофункциональный шаблон сайта для создания красивых электронных открыток.

## 📋 Содержимое проекта

- **index.html** - основной файл страницы
- **styles.css** - стили и оформление
- **script.js** - интерактивность и функции
- **README.md** - этот файл

## 🚀 Как использовать

### Локально на вашем компьютере

1. Откройте файл `index.html` в браузере (двойной клик на файл)

### На веб-сервере

1. Загрузите все три файла (index.html, styles.css, script.js) на ваш хостинг
2. Откройте index.html в браузере

## ✨ Возможности

- 🎨 **Выбор эмодзи** - 9 предустановленных вариантов (День рождения, свадьба, выпускной и т.д.)
- 📝 **Редактирование текста** - измените заголовок, сообщение и подпись
- 🎨 **Выбор цвета** - создавайте открытки любого цвета
- 📥 **Скачивание** - сохраните открытку как изображение
- 🔄 **Сброс** - вернитесь к исходным значениям
- 📱 **Адаптивный дизайн** - работает на телефонах, планшетах и компьютерах

## 🔧 Как модифицировать

### Добавить новое поздравление

В `index.html` в блоке `<select id="emojiSelect">` добавьте:

```html
<option value="🎉">🎉 Ваш случай</option>
```

### Изменить цвета

В `styles.css` измените градиент в классе `.greeting-card`:

```css
background: linear-gradient(135deg, #your-color1 0%, #your-color2 100%);
```

### Добавить новую функцию

Все функции обработки в `script.js`. Добавьте свой код после строки:

```javascript
// Helper functions
```
